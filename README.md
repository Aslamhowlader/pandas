## 📊 Useful Functions

| Function | Description |
|----------|-------------|
| `head()` | First 5 rows |
| `tail()` | Last 5 rows |
| `info()` | Dataset information |
| `describe()` | Statistical summary |
| `shape` | Rows & Columns |
| `columns` | Column names |
| `isnull()` | Missing values |
| `drop_duplicates()` | Remove duplicates |
| `fillna()` | Fill missing values |
| `sort_values()` | Sort data |
| `groupby()` | Group data |
| `to_csv()` | Save CSV |
